# 🚀 Console de Conteúdo - Versão Unificada

## 📋 Sobre

Console de Conteúdo é uma aplicação web unificada para gerenciar diferentes tipos de conteúdo:
- **Artigos** - Blog posts e artigos
- **VeloNews** - Notícias sobre ciclismo
- **Bot Perguntas** - Configurações para chatbot

## ✨ Características

- 🎨 **Interface moderna** com design responsivo
- 📱 **Totalmente responsivo** para mobile e desktop
- ⚡ **Performance otimizada** - tudo em uma página
- 🔄 **Console em tempo real** com logs de status
- 🎯 **Simplicidade** - sem complexidade de APIs

## 🛠️ Tecnologias

- **HTML5** - Estrutura
- **CSS3** - Estilização com gradientes e animações
- **JavaScript** - Funcionalidades e interações
- **Vercel** - Deploy e hospedagem

## 🚀 Deploy

O projeto está configurado para deploy automático no Vercel:

1. **Conecte o repositório** ao Vercel
2. **Deploy automático** a cada push
3. **URL pública** disponível instantaneamente

## 📁 Estrutura

```
/
├── index.html          # Página principal unificada
├── console.png         # Logo da aplicação
├── vercel.json         # Configuração do Vercel
└── README.md           # Este arquivo
```

## 🎯 Como Usar

1. **Acesse a aplicação** pela URL do Vercel
2. **Escolha o tipo** de conteúdo (Artigos, VeloNews, Bot Perguntas)
3. **Preencha o formulário** com as informações
4. **Envie os dados** - simulados para demonstração
5. **Acompanhe os logs** no console em tempo real

## 🔧 Desenvolvimento

Para desenvolvimento local:

```bash
# Clone o repositório
git clone [url-do-repo]

# Abra o index.html no navegador
# Ou use um servidor local:
python -m http.server 8000
```
